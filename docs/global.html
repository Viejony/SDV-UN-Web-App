setMapView()

Manages connection to the server and all interactions with ROS. Creates topic publishers and subscribers: - nav: draws a map in webpage - cancelTopic: send a cancel command to SDV - send2locationTopic: send a location and orientation message to SDV - actualLocationTopic: listens actual location of SDV and show it in the webpage. - teleop: sends velocity commands to SDV for manual operation.
